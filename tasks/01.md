## Этап 1: Базовый TCP echo-сервер

### Задача:

Написать TCP-сервер, который принимает входящие соединения и отправляет обратно клиенту всё, что тот написал (echo). Клиент подключается через `telnet` или `nc`.

### Требования:

1. Сервер слушает на порту `8080`
2. При подключении клиента — выводит в консоль сообщение вида `[+] New connection: 127.0.0.1:XXXXX`
3. Всё что клиент отправляет — сервер возвращает обратно с префиксом `echo: <сообщение>`
4. При отключении клиента — выводит `[-] Disconnected: 127.0.0.1:XXXXX`
5. Сервер не завершается после обрыва соединения — продолжает ждать новых клиентов
6. Корректно обрабатывает ошибку чтения (EOF при отключении клиента)

### Структура проекта:

```
tcp-chat/
    go.mod
    cmd/
        server/
            main.go
```

### Подсказки:

1. С чего начать: точка входа — `net.Listen("tcp", ":8080")`, затем цикл `listener.Accept()`. Каждый `Accept()` возвращает `net.Conn`.

2. Чтение строк: используй `bufio.NewReader(conn).ReadString('\n')` — читает до символа новой строки, что удобно для текстового протокола.

3. Адрес клиента: `conn.RemoteAddr().String()` возвращает строку `"127.0.0.1:PORT"`.

### Проверка работы:

`telnet localhost 8080` или `nc localhost 8080` — пишешь строку, получаешь `echo: <строка>` в ответ.
