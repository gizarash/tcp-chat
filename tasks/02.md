## Этап 2: Несколько клиентов одновременно

### Задача:

Сейчас сервер обслуживает клиентов последовательно — пока один подключён, второй ждёт. Нужно сделать так, чтобы сервер принимал несколько соединений одновременно, и каждый клиент получал свой echo независимо от других.

### Требования:

1. Сервер принимает несколько клиентов одновременно (проверяй двумя `telnet/nc` сессиями параллельно)
2. Каждый клиент обслуживается независимо — сообщения одного не мешают другому
3. Отключение одного клиента не влияет на остальных
4. В консоли сервера видны connect/disconnect всех клиентов

### Структура проекта:

```
tcp-chat/
    go.mod
    cmd/
        server/
            main.go
```

### Подсказки:

Одно ключевое слово `go handleConn(conn)`
