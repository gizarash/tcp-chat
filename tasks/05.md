## Этап 5: Команды

### Задача:

Добавить поддержку команд в чате. Когда сообщение начинается с / — это команда, а не обычное сообщение.

### Требования:

1. `/list` — отправляет клиенту список всех подключённых пользователей (по одному на строку)
2. `/msg <name> <message>` — приватное сообщение конкретному пользователю; получатель видит `[private] <from>: <message>`, отправитель получает подтверждение `[private → <name>]: <message>`
3. `/quit` — корректно отключает клиента (сервер делает то же, что при EOF)
4. Неизвестная команда — сервер отвечает клиенту: `unknown command: /xxx`
5. Обычные сообщения (не начинающиеся с `/`) работают как раньше

### Подсказки:

1. Парсинг команды: `strings.HasPrefix(input, "/")` проверяет, является ли строка командой. Для разбора аргументов удобно использовать `strings.SplitN(strings.TrimSpace(input), " ", 3)` — разбивает максимум на 3 части: `["/msg", "<name>", "<message>"]`.
2. Поиск клиента по имени: Придётся итерироваться по `s.clients` в поиске `net.Conn` с нужным именем. Это нормально для учебного проекта.
